# 🧮 Modern Calculator

A responsive calculator built using HTML, CSS, and JavaScript.

## 🚀 Features

- Basic arithmetic operations
- Decimal support
- Clear (AC)
- Delete (DEL)
- Keyboard support
- Responsive design
- Modern UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
