# 🧮 Aurora Calculator

A responsive calculator built using HTML, CSS, and JavaScript.

## 🌐 Live Demo

🔗 https://komaljawliya-tech.github.io/calculator-app/

## 🚀 Features

- Basic arithmetic operations
- Decimal support
- Clear (AC)
- Delete (DEL)
- Keyboard support
- Responsive design
- Modern UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
